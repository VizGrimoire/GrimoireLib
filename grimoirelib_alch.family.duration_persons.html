class grimoirelib_alch.family.duration_persons.DurationPersons(datasource, name, conditions=())[source]¶

Bases: grimoirelib_alch.family.common.Family

Factory of entities in the DurationPersons family.

This class can be used to instantiate entities from the DurationPersons family: those related to duration of persons.

Duration can be periods of different kinds, such as age or idle time.

Objects of this class provide the function durations() to obtain an ActorsDuration object.

Methods

durations() Return durations for each person (age, idle,...) depending on entity
set_activity(activity) Change the activity list considered by the entity.
set_snapshot(time) Define a snapshot time for durations.

class grimoirelib_alch.family.duration_persons.SnapshotCondition(date)[source]¶

Bases: grimoirelib_alch.family.duration_persons.Condition

Condition for specifiying “snapshot” of durations.

Durations (age, idle) are to be calculated back from the date specified by this condition. If the snapshot date is as of now, this will be the real “age” or “idle time” in the project.
