class grimoirelib_alch.family.scm.SCM(datasource, name, conditions=(), echo=False)[source]¶

Bases: grimoirelib_alch.family.common.DBFamily

Constructor of entities in the SCM family.

This class can be used to instantiate entities from the SCM fammily: direct entities obtained querying a CVSAnalY database.

The entity to be instantiated is specified using its name, and a set of zero or more conditions that provide context.

This class provides functions to return different kinds of aggregation (timeseries, total, list) and selection (dates, etc.).

Valid entity names: “ncommits”, “listcommits”, “nauthors”, “listauthors”
